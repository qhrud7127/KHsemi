# 삼부초등학교


 ## 삼부초등학교 소개
 
> ###  프로젝트 명
>
> 삼부초등학교 홈페이지 구현
>
> ### 개발 목적
>
>>초등학생, 학부모, 교사 등 사용 연령층은 다양하지만
>>
>>다른 홈페이지에 비해 비교적 낙후된 요즘 초등학교 홈페이지를 
>>
>>사용자가 조금 더 편리하게 사용할 수 있도록 개선  
>>
>
>
> ### 프로젝트 기대효과
>
>>    다양한 연령대가 사용하기에도 문제없는 직관적인 디자인으로 사용자의 편의성을 높임
>>
>>    도서관에 방문하더라도 원하는 책을 빌리지 못하는 경우가 있다. 
>>
>>    모바일 연동이 안되던 기존 홈페이지의 불편함을 고려해 반응형 웹사이트로 모바일과 태블릿 버전 지원
>>
>
>


>
> ## 개발언어 및 도구
>
>>  OS : Windows10
>>
>> Tool : eclipse, Brakets, SQLyog
>>
>> Language : JAVA , HTML, CSS, JS, jQuery
>>
>> DB : MySQL
>>
>> Server : Apache Tomcat 
>>
>> CSS : Bootstrap
>>
